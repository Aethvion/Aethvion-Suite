Historical Runs

How the Suite Tester works: When you press "Run Automatic Tests", the devtool backend spawns a separate instance of the Aethvion Suite backend on port 18080. It then automatically launches a browser tab pointing to it. The tool profiles both backend process footprints (CPU & Memory) and system-wide telemetry at 1-second intervals for 60 seconds. At second 15, an LLM task load is injected to verify agent routing speed under render stress.
